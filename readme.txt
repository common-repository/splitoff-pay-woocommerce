Provide TogetherPay as a payment option for WooCommerce orders.

== Description ==

Give your customers the option to buy together with TogetherPay. The "TogetherPay for WooCommerce" plugin provides the option to choose TogetherPay as the payment method at the checkout. It also provides the functionality to display the SplitOff logo below product prices on category pages, individual product pages, and on the cart page. For each payment that is approved by TogetherPay, an order will be created inside the WooCommerce system like any other order. Automatic refunds are also supported.

== Installation ==

This section outlines the steps to install the SplitOff plugin.

> Please note: If you are upgrading to a newer version of the TogetherPay plugin, it is considered best practice to perform a backup of your website - including the WordPress database - before commencing the installation steps. SplitOff recommends all system and plugin updates to be tested in a staging environment prior to deployment to production.

1. Login to your WordPress admin.
1. Navigate to "Plugins > Add New".
1. Type "TogetherPay" into the Keyword search box and press the Enter key.
1. Find the "TogetherPay for WooCommerce" plugin. Note: the plugin is made by "SplitOff".
1. Click the "Install Now" button.
1. Click the "Activate" button.
1. Navigate to "WooCommerce > Settings".
1. Click the "Checkout" tab.
1. Click the "TogetherPay" sub-tab.
1. Enter the Merchant ID and Secret Key that were provided by SplitOff for Production use.
1. Save changes.
